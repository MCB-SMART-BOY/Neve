### 1. Zero Ambiguity (é›¶äºŒä¹‰æ€§)

Every syntax construct has one and only one meaning. No context needed. No guessing.

```neve
-- In Neve, you ALWAYS know what you're looking at:

#{ x = 1 }       -- This is ALWAYS a record. Period.
{ let x = 1; x } -- This is ALWAYS a block.
```

Compare to Nix:
