# English

> A pure functional language for system configuration and package management.

## The Story

*"I love Nix. I really do. But every time I write `{ x = 1; }` and wonder if it's a record or a function... I die a little inside."*

That's how Neve was born. Not from hatred, but from love â€” and a desire to do better.

Neve takes everything beautiful about Nix (pure functional, reproducible, declarative) and wraps it in syntax that doesn't make you flip tables. It's not a Nix replacement. It's not compatible with nixpkgs. It's a **clean-slate reimagination**.

